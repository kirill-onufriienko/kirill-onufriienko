## What I've been building

### Trading Analytics Pipeline
Working with raw trading logs, cleaning and validating data, calculating key metrics, and turning the results into clear Excel reports.

**Built with:** Python, pandas, openpyxl, CSV, Excel  
**Repo:** [View project](https://github.com/kirill-onufriienko/trading-analytics-pipeline)

### SQL Reporting Pipeline

**Built with:** Python, PostgreSQL, SQLAlchemy, psycopg2, SQL, Excel
